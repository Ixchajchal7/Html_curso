Curso introductorio a HTML y CSS

En este curso aprendemos como utilizar HTML y CSS: los lenguajes de la web.

  • Manual Oficial de la W3C

HTML: Hypertext Markup Language

Es un lenguaje compuesto por etiquetas, las cuales describen elementos. En el navegador web es el encargado de interpretar el codigo y mostrarlo de forma comprensible a nuestros usuarios.

Junto con CSS y Javascript. HTML es parte de la triada de languajes que definen nuestra Web.

Elementos HTML y semantica

Los elementos HTML son bloques de construccion para la maqueta de nuestra pagina Web. Los elementos HTML nos permiten incorporar texto, imagenes, vinculos, entre otros. HTML tambien proporciona un contexto semantico de elementos el cual describe su proposito, como por ejemplo titulos, parrafos, listas, enlaces, citas y otros.

CSS, Cascade Styling Sheets

Hojas de estilo en cascada

CSS es el lenguaje que nos permite agregarle estilos a nuestro documento HTML. CSS describe como los elemenetos HTML deber ser vistos en el navegador. En este link podemos acceder al manual oficial de CSS
